# To-Do Mobile App

Mobilna aplikacija za upravljanje zadacima razvijena pomoću React Native tehnologije za potrebe učenja.

## Opis projekta
Aplikacija omogućava korisnicima da:
- dodaju zadatke
- obrišu zadatke

## Tehnologije
- React Native
- JavaScript

## Pokretanje aplikacije

1. Klonirati repozitorijum
2. Instalirati zavisnosti
